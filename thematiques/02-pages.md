# Pages
Les *pages* ou écrans s’articulent autour de 3 axes principaux:
- Layout     : comment *dessiner* des éléments pour composer un écran
- Évènements : comment réagir aux volontés de l’utilisateeur
- Navigation : comment passer d’un écran à l’autre
